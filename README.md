# Professional-Calculator-Python
A robust, command-line scientific calculator built with Python. Features advanced math (Factorial/Gamma), persistent history management, and comprehensive error handling

## Key Features
* *Advanced Calculations:* Includes standard operations plus Factorial and Gamma functions.
* *Smart History Management:* Track, clear, and manage your calculation history seamlessly.
* *Robust Error Handling:* Prevents crashes from invalid inputs or mathematical errors.
* *Dynamic Input System:* Support for multiple numerical inputs in a single session.

## Technical Skills Demonstrated
* Python Standard Library (math module)
* Data Structures (Lists & Dictionaries)
* Exception Handling (try-except blocks)
* Modular Function Design

## How to Use
1. Clone the repository.
2. Run professional_calculator.py using Python 3.
3. Follow the on-screen menu to perform calculations.
